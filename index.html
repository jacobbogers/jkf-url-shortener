URL shortener Microservice

Usage Examples:
  • To shorent a link-url append after the link of this site after the word "new"

    https://jkf-url-shortener.herokuapp.com/new/http://google.com

  • To use a short link just enter the shortened link in your browser shorent a link-url append after the link of this site

    https://jkf-url-shortener.herokuapp.com/0123456

On Success will return:

{ orig: "/new/http://www.facebook.com", base: "https://jkf-url-shortener.herokuapp.com/1244" }

On Error will return:

{ error: "This url: [https://jkf-url-shortener.herokuapp.com/1244] is not in the database." }
